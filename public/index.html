Esta es mi API

¿Como empezar a usar la API?

Registro

Para comenzar a utilizar la API tenes que registrarte usando la ruta localhost:3000/usuarios/registrar por POST y escribir en el body un nombre, un email y una contraseña, acordate de esto porque por el momento no existe manera de recuperarlos, así que recomiendo anotarlos.

Login

Para comenzar a hacer consultas en la API tenes que loguearte usando la ruta localhost:3000/usuarios/login por POST y escribir en el body el email y contraseña que utilizaste en tu registro, una vez registrado te daran un token, este guardalo para poder hacer las peticiones, para empezar a hacer peticiones que requieren autorizacion tienes que ir a la parte de authorization en postman y seleccionar bearer token, luego meter el token que te han dado y así podras empezar a utilizar la API con todas sus funciones.

Obtener todos los usuarios

Para obtener una lista completa de usuarios registrados, utiliza la ruta: localhost:3000/usuarios/ con el método GET. Esta ruta requiere autenticación, por lo que debes enviar un token en los headers.

Actualizar un usuario por su ID

Para actualizar los detalles de un usuario, utiliza la ruta: localhost:3000/usuarios/:id con el método PUT. Debes estar autenticado y enviar los datos actualizados en el cuerpo de la petición.

Eliminar un usuario por su ID

Para eliminar un usuario, utiliza la ruta: localhost:3000/usuarios/:id con el método DELETE. Al igual que otras operaciones, necesitas autenticación para eliminar usuarios.

Obtener un usuario por su ID

Para obtener un usuario específico por su ID, utiliza la ruta: localhost:3000/usuarios/:id con el método GET. Esta ruta también requiere autenticación.

¿Cómo usar las rutas de libros?

Obtener todos los libros

Para obtener una lista completa de todos los libros, utiliza la ruta: localhost:3000/libros/ con el método GET. No necesitas autenticación para esta consulta.

Obtener libros con paginado

Si quieres obtener libros en formato paginado, utiliza la ruta: localhost:3000/libros/paginado con el método GET. Esto te permitirá consultar los libros en partes, útil para grandes colecciones.

Buscar libros por título

Para buscar libros por su título, utiliza la ruta: localhost:3000/libros/buscar/nombre?nombre=El%20Principito con el método GET. Asegúrate de proporcionar el nombre del libro en los parámetros de consulta.

Crear un nuevo libro

Para agregar un nuevo libro, utiliza la ruta: localhost:3000/libros/ con el método POST. Debes estar autenticado, así que asegúrate de enviar el token de autorización en los headers como Bearer token. Ejemplo del cuerpo (body) de la petición: 


{
  "titulo": "El nuevo libro",
  "autor": "Juan Pérez",
  "genero": "Ficción",
  "anio_publicacion": 2024
}

Buscar libros por género

Si deseas buscar libros por género, usa la ruta: localhost:3000/libros/buscar/genero con el método POST. Debes incluir el género en el cuerpo (body) de la petición, de la siguiente forma: 


{
  "genero": "Ciencia ficción"
}

Obtener un libro por ID

Para obtener un libro específico por su ID, utiliza la ruta: localhost:3000/libros/:id con el método GET. Solo necesitas reemplazar :id por el ID real del libro que deseas consultar.

Actualizar un libro por ID

Para actualizar los detalles de un libro específico, utiliza la ruta: localhost:3000/libros/:id con el método PUT. Al igual que para crear un libro, debes estar autenticado (token de autorización). En el cuerpo de la petición, proporciona los nuevos datos del libro que deseas actualizar.

Eliminar un libro por ID

Para eliminar un libro por su ID, utiliza la ruta: localhost:3000/libros/:id con el método DELETE. Recuerda que necesitas autenticación para esta operación.

¿Cómo usar las rutas de autores?

Obtener todos los autores

Para obtener una lista completa de todos los autores, utiliza la ruta: localhost:3000/autores/ con el método GET. No necesitas autenticación para esta consulta.

Crear un nuevo autor

Para agregar un nuevo autor, utiliza la ruta: localhost:3000/autores/ con el método POST. Debes estar autenticado, así que asegúrate de enviar el token de autorización en los headers como Bearer token. Ejemplo del cuerpo (body) de la petición: 


{
  "nombre": "Gabriel García Márquez",
  "nacionalidad": "Colombiana",
  "libros": ["Cien años de soledad", "El amor en los tiempos del cólera"]
}

Buscar autores por nombre

Para buscar autores por su nombre, utiliza la ruta: localhost:3000/autores/buscar/nombre?nombre=Gabriel con el método GET. Asegúrate de proporcionar el nombre del autor en los parámetros de consulta.

Actualizar un autor por ID

Eliminar un autor por ID

Para eliminar un autor por su ID, utiliza la ruta: localhost:3000/autores/:id con el método DELETE. Recuerda que necesitas autenticación para esta operación.

Obtener un autor por ID

Para obtener un autor específico por su ID, utiliza la ruta: localhost:3000/autores/:id con el método GET. Solo necesitas reemplazar :id por el ID real del autor que deseas consultar.
